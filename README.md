# Product Description 
This web app is an admin dashboard that shows all the users, transcations, total revenue, total sales and  total cost. 
Admin can also perfom crude operations

# Features
- Add new user
- Edit user
- Update user
- Delete user
- Add new product
- Edit product
- Delete product

# Technologies Used
 - React js
 - CSS
 - Rechart
 - Material UI
